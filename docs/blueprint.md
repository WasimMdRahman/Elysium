# **App Name**: Zenith Mind

## Core Features:

- Walkthrough: A clean walkthrough page detailing the app and its features with a next button after each feature, leading to the sign-up page.
- Dashboard: Interactive dashboard with six feature cards, username display, and a greeting. Cards pop out on hover.
- Side Panel: Persistent side panel for navigation: Dashboard, Streak Tracker, Settings, Privacy Policy, Terms of Service, and Pricing.
- AI Chatbot: 24/7 mental health support providing responses in professional, friendly, empathetic, or humorous tones. It is a tool that uses chat summarization, voice input/output, emotion detection, and chat history.
- Mood Tracker (Emotion Quest): Daily mood logging (scale of 1-10) displayed in a weekly line graph with date labels.
- AI Journal: Private journal with rename/delete options and entry history (date/time).
- Task Manager: Daily to-do list management with task creation and completion tracking.
- Thought Quest Game: Interactive CBT-based game with AI-generated thoughts (helpful/unhelpful classification). Animation feedback and score tracking. Generative AI used to generate new thoughts.
- Mental Health Exercises: Library of guided mental health exercises categorized by challenges.
- Daily Streak Tracker: Streak tracking for core activities, XP/AP rewards, and medal achievements.

## Style Guidelines:

- Background color: Very light blue-gray (#F0F4F8) for a calm, unobtrusive feel.
- Primary color: Muted blue (#748DA6) to inspire calmness and trust.
- Accent color: Soft lavender (#B19CD9) for highlights and calls to action, analogous to blue, offering a contrast in saturation and brightness, yet remaining gentle.
- Body and headline font: 'PT Sans', a humanist sans-serif to be used throughout the application for a modern, calm feel.
- Use clean, minimalist icons in a single color (the primary color) for a consistent and calming visual experience.
- Dashboard will be a grid of six visually distinct cards, each representing a core feature. Interactive cards will show details via modal overlay or 'pop-out'.
- Subtle animations for feedback on user interactions.
